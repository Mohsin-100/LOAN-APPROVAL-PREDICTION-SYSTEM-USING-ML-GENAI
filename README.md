# LOAN-APPROVAL-PREDICTION-SYSTEM-USING-ML-GENAI
An end-to-end Loan Prediction System integrating Machine Learning with Google Gemini 3.5 Flash to provide AI-powered explanations, financial guidance, loan reports, and intelligent banking assistance.
**OVERVIEW:**
Traditional Machine Learning models are excellent at making predictions, but they often fail to explain why a prediction was made.

This project demonstrates how Generative AI can be integrated on top of a Machine Learning model to bridge that gap.

The application predicts whether a loan should be approved and then uses Google Gemini 3.5 Flash to generate human-friendly explanations, financial advice, reports, and customer communication.

This project was developed as part of a teaching initiative to help students understand how Machine Learning and Generative AI complement each other in real-world business applications.
